# 一.标题头
```
---
title:
description:
categories:
photos:
date:
tags:
---
```
分别代表着：标题、描述、分类、图片、时间和标签，其中分类要使用
```
categories:
 - X
```
注意写作过程中使用英文的“:”号，并且保证“:”后空一格填写信息

## (三) 文字样式
### 加粗
使用`**加粗**`或者`__加粗__`来对文字进行**加粗**
### 斜体
使用`*斜体*`或者`_斜体_`来对文字进行*斜体*
### 加粗&斜体
使用`***加粗&斜体***`、`___加粗&斜体___`、`**_加粗&斜体_**`、`__*加粗&斜体*__`来对文字进行***加粗&斜体***

## (四) 引用
单块引用使用`>`在段落前
`> 单一段落引用`
> 单一段落引用

### 多个段落引用
在段落空行之间添加一个`>`

```
> 多个段落进行引用
>
> 这是第二段引用
```

> 多个段落进行引用
>
> 这是第二段引用

### 嵌套引用
引用可以嵌套，需要在嵌套引用的段落前添加`>>`

```
>这是第一层引用
>
>>这是二层引用
```

>这是第一层引用
>
>>这是二层引用

## (五) 列表

### 有序列表
数字句号空格之后输入，数字不一定按顺序，但是要以1开头
```
1. 第一项
2. 第二项
3. 第三项
    1. 缩进项目
    2. 缩进项目
4. 第四项
8. 第五项
```

1. 第一项
2. 第二项
3. 第三项
    1. 缩进项目
    2. 缩进项目
4. 第四项
8. 第五项

### 无序列表
在项目前添加段划线`-`、星号`*`或加号`+`
```
* 第一项
- 第二项
+ 第三项
* 第四项
  * 缩进项目
  * 缩进项目
* 第五项
```

* 第一项
- 第二项
+ 第三项
* 第四项
  * 缩进项目
  * 缩进项目
* 第五项
